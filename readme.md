# 김서연 | Portfolio Website

모던하고 세련된 개인 포트폴리오 웹사이트입니다.

## ✨ Features

- **풀 페이지 원페이지 디자인** - 프로필, 스킬, 포트폴리오, 연락처를 하나의 페이지에 담았습니다
- **반응형 디자인** - 모바일, 태블릿, 데스크탑 모든 환경에서 최적화
- **타이핑 애니메이션** - 동적인 타이틀 효과
- **스크롤 애니메이션** - IntersectionObserver 기반 스크롤 트리거 애니메이션
- **Netlify Forms** - 서버 없이 이메일 전송 가능한 컨택트 폼

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)

## 🚀 Deployment

Netlify에 자동 배포됩니다.

## 📬 Contact Form

Netlify Forms를 사용하여 별도의 백엔드 서버 없이 폼 전송이 가능합니다.
